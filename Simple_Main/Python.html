Syntax

Python uses indentation to define code blocks. Statements end with a newline, not a semicolon.

            if True:
                print("Indented block")

Comments

Use # for single-line comments and ''' ... ''' for multi-line comments.

            # This is a comment
            '''
            This is a
            multi-line comment
            '''

Variables

Variables store data. No need to declare type:

            x = 5
            name = "Alice"

Data Types

Common types: int, float, str, bool, list, tuple, set, dict.

            a = 10      # int
            b = 3.14    # float
            c = "text"  # str
            d = True    # bool

Numbers

Python supports integers, floats, and complex numbers.

            a = 10
            b = 3.14
            c = 2 + 3j

Casting

Convert between types using int(), float(), str():

            x = int(3.5)
            y = str(10)

Strings

Strings are sequences of characters. Use single or double quotes.

            text = "Hello"
            print(text[0])  # H

Booleans

Boolean values are True and False.

            is_active = True

Operators

Python supports arithmetic, comparison, logical, assignment, and more operators.

            a = 5 + 3
            b = a > 2
            c = a and b

Lists

Lists store multiple items:

            fruits = ["apple", "banana", "cherry"]
            print(fruits[1])

Tuples

Tuples are like lists but immutable:

            coordinates = (10, 20)

Sets

Sets store unique items:

            colors = {"red", "green", "blue"}

Dictionaries

Dictionaries store key-value pairs:

            person = {"name": "John", "age": 30}
            print(person["name"])

If...Else

Conditional statements:

            x = 10
            if x > 5:
                print("x is greater than 5")
            else:
                print("x is 5 or less")

While Loops

Repeat code while a condition is true:

            count = 0
            while count < 5:
                print(count)
                count += 1

For Loops

Iterate over sequences:

            for i in range(5):
                print(i)

Break

Exit a loop early:

            for i in range(10):
                if i == 5:
                    break
                print(i)

Continue

Skip to the next iteration:

            for i in range(5):
                if i == 2:
                    continue
                print(i)

Range

Generate a sequence of numbers:

            for i in range(1, 6):
                print(i)

Functions

Define reusable code blocks:

            def greet(name):
                print("Hello, " + name)

            greet("Alice")

Lambda

Anonymous functions:

            add = lambda a, b: a + b
            print(add(2, 3))
